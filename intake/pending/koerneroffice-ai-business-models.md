## Card 1 -- Sell CLAUDE SKILLS as a digital product (STRONGEST for the portfolio)
Source video: quYKZushRPo (guest Ryan Dozer, a one-person 6-figure marketing agency).
Model: package your subject-matter expertise as a stack of Claude skills (markdown SOPs,
one skill per task) and sell it as a digital product. He sells a ~20-25 skill "Claude Code
skill stack" for $99 and showed a Stripe dashboard of $3,000+ in 30-45 days (UNVERIFIED),
plus a $1,000/hour consulting upsell that closed on the spot. Three monetization paths from
ONE asset: (a) sell the skill pack, (b) use the skills to deliver higher-quality client work
(the skills ARE his agency's edge), (c) skill-led consulting.
How (non-technical): voice-dump your expertise into the Claude app -> run Anthropic's free
skill-creator skill -> get a skill markdown file -> package + sell. "Everyone has expertise
in something; no one is exempt." Distribution is self-hosted: he vibe-coded the landing page
with his own "web designer skill" + a Stripe link in ~15-20 min, then drives it with a
floating popup, blog CTAs, and email drips (dog-fooding -- the skills build the site that
sells the skills). Granularity matters: one SOP per subtask (SEO blog writer, email writer,
thumbnail designer, anti-slop skill), not one giant skill.
Requires: subject-matter expertise (have it), Claude subscription, a Stripe account, a
landing page (vibe-coded). Near-zero capital; the cost is curation time.
FIT: HIGH / most portfolio-relevant model on either menu. The operator's forge IP is
LITERALLY a Claude-skills / Claude-mastery curriculum -- claude-techniques skills could be
packaged and sold exactly this way, and the PrepBrix teaching method could become a sellable
"NEET/JEE study-system" skill pack. This is the cleanest path yet seen to productize the
forge IP. GRADUATED 2026-06-29 -> VentureForge/intake/pending/forge-ip-claude-skill-pack.md
(scored the strongest secondary-venture candidate seen; gated NOT-A-PLAY-YET pending two
operator actions -- adopt the secondary venture in GOALS.md + set a dated milestone).
Technical nuggets worth keeping (see ConceptForge REFERENCES): Anthropic ships a free
skill-creator skill on GitHub (best-practice scaffold, better than "make a skill"); skills
are transferable across agents (Claude Code -> Codex -> Gemini) because they are markdown
(the AGENTS.md portability point in claude-md-config); skills self-correct (fix once -> the
skill is updated so the mistake never repeats), with auto-skill-updating in the background
("Karpathy auto-research"); models read markdown / HTML / txt far more efficiently and
accurately than PDFs. His "anti-slop skill" bans the same kind of filler words and the
"not X, but Y" construction this pipeline does -- independent corroboration of the anti-slop gate.

## Card 2 -- Missed-call-text-back AI agent for SMBs (DEDUP + one fresh mechanic)
Source video: g4-3AxelI_Y. Model: build a "missed call -> AI texts back -> two-way
conversation -> books the appointment" agent in GoHighLevel and sell it to local service
businesses (tree care, plumbing, HVAC) for $500-2,000/mo per client; the conversation
prompt is written with Claude. Anchored on the real "speed-to-lead" stat that the first
responder wins most of the sale.
FIT: DEDUP -- this is the same SMB voice/text-agent category as Silicon Valley Girl Card 3
(voice AI receptionist) and the recurring AI-to-SMB-services theme. OFF the education
portfolio. The ONE fresh, reusable mechanic worth keeping: DOG-FOODING for client
acquisition -- use the very same missed-call/ringless-voicemail + AI-text-back system in
REVERSE to find and close the SMB clients you will sell it to (scrape business numbers with
Outscraper -> ringless voicemail drop via a broadcast tool -> AI texts back the call-backs
and pitches). "Use the product to acquire customers for the product" is a clean distribution
idea that generalizes beyond this niche.
